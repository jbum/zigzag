# Rules.md

clue_finish_a	If a clue has sufficient non-meets, then fill in it’s meets.
clue_finish_b	If a clue has sufficient meets, then fill it’s avoiders.
no_loops	If placing a slant in an unknown square creates a loop, then place the counter slant.
forced_solution_avoidance	If providing the final slant to a given would cause an adjacent given to be unsolvable, it cannot be in that square.
loop_avoidance_2	if finishing a 2 with two slants would force a loop, then the loop must be finished in some other way.
